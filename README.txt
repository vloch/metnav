README
======

metnav product for plone

This product provides :
- facet navigation on LOM(Learning Objects Metadata)  metadata resources stored in a repository responding to xQuery request,
- management of related documents to the displayed document,
- capability to choose the outstanding document suggested,
- capability to display the list of last documents whose metadata have been written in the database.
 This product is best fitted with existDA product that brings a connector to eXist-db XML native database.

This product can run on eXist v.1 and v.2 databases.

Configuration
-------------
metnav comes with a backoffice configuration system that enables to raise a dedicated form which enables to manage connections to the repository and other parameters.
